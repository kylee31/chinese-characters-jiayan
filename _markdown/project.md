## 프로젝트명: 자옌(宋甲)

- 고전 중국어 전문 파이썬 자연어 처리 도구(python)를 사용하여 기존의 중국어 자연어 처리 도구들은 주로 현대 중국어 데이터로 학습되어 고전 중국어 처리에는 성능이 저조한 상황
- 본 프로젝트에서 고전 중국어 정보 처리를 지원하는 웹서비스를 구축하는 것을 목표로 함
- 현재 기능은 1. 어휘 구축, 2. 토큰화, 3. 품사 태깅, 4. 문장 분할, 5. 자동 구두점 총 5개 지원

---

### 참고

모델에 사용된 데이터의 특성상 현재는 번체 중국어를 지원하지 않음. 번체 중국어를 처리해야 하는 경우, [OpenCC](https://github.com/yichen0831/opencc-python)를 사용하여 입력값을 간체자로 변환한 후 결과를 다시 번체로 변환해야 함. 부가 기능으로 OpenCC를 활용하여 번체 > 간체 변환 서비스를 제공할 계획이 있음

---

#### 1. 어휘 구축

비지도 학습 방식을 사용하여 트라이 트리, PMI ( 점별 상호 정보 ) 및 좌우 문자의 인접 엔트로피를 포함하는 어휘집 구축

#### 2. 토큰화

사전을 사용하지 않는 비지도 학습 방식을 통해 N-gram 언어 모델과 HMM ( 은닉 마르코프 모델 ) 을 이용하여 고전 중국어 문장 토큰화
어휘 구축을 통해 생성된 사전을 사용하여 방향성 비순환 단어 그래프, 최대 확률 경로 및 동적 프로그래밍 으로 고전 중국어 문장 토큰화

#### 3. 품사 태깅

CRF ( 조건부 난수 생성기 ) 를 사용한 단어 수준 시퀀스 태깅

#### 4. 문장 분할

CRF를 이용한 문자 수준 시퀀스 태깅은 PMI 및 T-검정 값을 특징으로 도입

#### 5. 자동 구두점

계층형 CRF를 사용한 문자 수준 시퀀스 태깅을 통해 문장 분할 결과를 기반으로 주어진 고전 중국어 텍스트에 구두점 부여
